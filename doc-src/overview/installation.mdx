# 安装

通过CDN直接使用：

```html
<script src="https://unpkg.com/glsl-doodle/dist/glsl-doodle.js"></script>
```

通过NPM安装

```bash
npm i glsl-doodle --save
```

通过Yarn安装

```bash
yarn add glsl-doodle --save
```
